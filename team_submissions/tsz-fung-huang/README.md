# Google Stock Trading Strategy - Project Documentation

## Project Structure

```
coursework/
├── backtest_multi_position.py    # Main backtesting script
├── statistics.py                  # Statistical analysis script
├── googl_processed_data.parquet   # Processed data with indicators
├── googl_ohlcv.parquet            # Raw OHLCV data (optional)
```

## Quick Start

### Prerequisites
```bash
pip install pandas numpy matplotlib seaborn yfinance talib scipy pyarrow
```

### Running the Backtest
```bash
python backtest_multi_position.py
```

This will:
1. Download GOOGL data from Yahoo Finance (2015-2025)
2. Calculate all technical indicators
3. Run the two-position backtest
4. Generate performance metrics
5. Save visualization to `backtest_two_position_comparison.png`
6. Save processed data to `googl_processed_data.parquet`

### Running Statistical Analysis
```bash
python statistics.py
```

This requires `googl_processed_data.parquet` to exist (generated by backtest script).

## Main Scripts

### backtest_multi_position.py
Main backtesting script implementing:
- Data download and processing
- Technical indicator calculations (RSI, EMA, CRSI, Trend classification)
- Two-position trading strategy
- Performance metrics calculation (Sharpe Ratio, Win Rate, Annualized Return, Max Drawdown)
- Visualization

### statistics.py
Statistical analysis script providing:
- CRSI 2D bucket analysis
- Trend-based hit rate analysis
- Scenario-specific analysis
- Advanced strategy metrics
- Multiple visualizations

## Notes

- All prices are in USD
- Data automatically adjusted for splits and dividends
- Strategy uses daily closing prices only
- Transaction costs not included in backtest
- Results are historical and may not predict future performance
